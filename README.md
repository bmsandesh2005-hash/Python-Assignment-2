# Python Assignment 2

This repository contains simple Python programs demonstrating basic programming concepts such as loops, conditionals, and user input.

## Task 1: Even or Odd Checker

**Description:**  
This program takes an integer input from the user and checks whether the number is even or odd using an `if-else` statement.  
It then displays the result accordingly.

**Concepts Used:**
- User input
- `if-else` condition
- Modulus operator (`%`)

## Task 2: Sum of Numbers from 1 to 50

**Description:**  
This program uses a `for` loop to iterate through numbers from 1 to 50.  
It calculates the sum of all integers in this range and displays the final result.

**Concepts Used:**
- `for` loop
- `range()` function
- Integer addition
